Qu'est ce que Knight NPC Generator ?

Knight NPC Generator est un outil de création de PNJ pour le jeu de rôle Knight des éditions Antre-Monde. Vous trouverez toutes les informations nécessaires sur le site officiel : https://knight-jdr.fr/

Comment partiper au projet ?

Si vous en avez les compétences et que vous désirez participer au projet, vous pouvez retrouver le projet sur github.

Comment signaler un bug ou faire une suggestion ?

Le moyen le plus efficace est de créer un ticket sur projet github. Vous pouvez aussi retrouver le sujet qui parle de Knight NPC generator sur les forums officiels du jeu de rôle Knight dans la partie réservée aux MJ.
